# CASRAD User Guide

## Getting Started

### First Login

1. Navigate to {SERVER_URL}
2. Complete the setup wizard (first run only)
3. Create your admin account
4. Start uploading or configuring music directories

### Managing Your Library

#### Adding Music

1. Go to Settings → Library
2. Add music directories
3. Click "Scan Library"

#### Creating Playlists

1. Browse your library
2. Click "+" next to tracks to add to queue
3. Save queue as playlist

### Streaming Protocols

#### MPD Protocol

Connect any MPD client to:
- **Host**: {SERVER_URL}
- **Port**: {MPD_PORT}

#### Subsonic API

Configure your app with:
- **Server**: {SERVER_URL}/subsonic
- **Username**: Your CASRAD username
- **Password**: Your CASRAD password

#### Ampache API

Configure your app with:
- **Server**: {SERVER_URL}/ampache
- **Generate API key**: Settings → API Tokens

## Mobile Apps
