# CASRAD Documentation

Welcome to CASRAD - Complete Audio Streaming, Radio, and Distribution.

## Quick Start

CASRAD is designed to work with zero configuration. Simply run the binary:

```bash
./casrad
```

Open your browser to the URL shown (default: http://localhost:64000)

## Server Details

- **Server URL**: {SERVER_URL}
- **MPD Port**: {MPD_PORT}
- **RTMP Port**: 1935
- **Admin Email**: {ADMIN_EMAIL}

## Features

- **MPD Protocol**: Connect with any MPD client
- **Subsonic API**: Use Subsonic-compatible apps
- **Ampache API**: Connect with Ampache clients
- **WebDAV**: Access your music via WebDAV
- **RTMP Streaming**: Live broadcasting support
- **DLNA/UPnP**: Stream to DLNA devices

## Configuration

All configuration is done through the web interface at {SERVER_URL}/admin
